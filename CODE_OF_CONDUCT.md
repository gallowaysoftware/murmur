# Contributor covenant code of conduct

This project follows the
[Contributor Covenant v2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/).

In short: be kind, be patient, and assume good faith. Disagreement is fine
(adversarial code review is a feature here, not a bug); contempt is not.

## Enforcement

Reports go to `conduct@gallowaysoftware.ca`. We aim to acknowledge within 3
business days. Reporters' identities are kept confidential to the maximum
extent compatible with addressing the issue.

Possible outcomes range from a private clarification through to a temporary
or permanent ban from the project's spaces. The maintainers' decisions are
final.

## Scope

This applies to all project spaces — issues, PRs, discussions, code review
comments, the project's social media accounts, and any conference / meetup
space where someone is representing the project.
